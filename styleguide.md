# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1366px

## Colors

### Primary

- Blackberry: hsl(266, 8%, 18%)
- Rusty: hsl(24, 74%, 51%)
- Pink Guava: hsl(354, 63%, 60%)
- Peach: hsl(1, 87%, 91%)
- Petal: hsl(43, 100%, 99%)

### Light and Dark Mode

- Peach:
    Button Text

- Pink Guava:
    Logo Icon
    Button gradient start

- Rusty:
    Button gradient end

### Light Mode

- Petal:
    Background
    Button Text

- Blackberry:
    Text

### Dark Mode

- Petal:
